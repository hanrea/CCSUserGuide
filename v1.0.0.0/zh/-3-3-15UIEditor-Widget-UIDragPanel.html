拖拽层[UIDragPanel]

图标

描述

UIDragPanel的类是一个UIWidget子类，它实现了在触摸屏上的拖拽层。

触摸一个拖拽层时拦截事件和动作消息发送到目标对象时。设定的目标和行动方法都继承自UIWidget。该控件提供了方法来设置拖拽层的裁切，显示区域宽高，拖拽区域宽高，背景色透明度，填充颜色，图片资源等属性。

模拟裁剪

 

隐藏掉控件显示区域外的子控件

 

文件

 

配置一张默认下的图片（png,jpg），控件默认状态下显示。

 

滚动区域宽

 

设置拖拽区域的宽度。

 

滚动区域高

 

设置拖拽区域的高度。

 

 

注意：滚动区域宽高设置后渲染区会有一个绿色的线框与之对应，需要拖拽的子控件都必须包涵在绿色线框内才能进行拖拽。

高级属性

惯性持续时间

 

惯性持续时间是指当停止对拖拽层的拖拽操作时，拖拽层根据惯性继续移动所持续的时间。

 

惯性缓动频率

 

惯性缓动频率指指当停止对拖拽层的拖拽操作时，拖拽层根据惯性继续移动时的速度变化频率，默认值为2。当该值大于1时，拖拽层后惯性移动是越来越慢的，值越大，减速效果越明显；当该值小于1时，拖拽层后惯性移动是越来越快的，值越大，加速效果越明显。

 

开启回弹

 

设置拖拽层是否开启回弹效果。若开启，拖拽层的内容将能够被拖动至脱离拖拽层边界。

持续时间

 

持续时间是指当停止对拖拽层的拖拽操作时且拖拽层内容脱离边界时，拖拽层内容移动至拖拽层边界（与拖拽方向相反）所持续的时间。

 

缓动频率

 

缓动频率是指当停止对拖拽层的拖拽操作时且拖拽层内容脱离边界时，拖拽层内容向拖拽层边界（与拖拽方向相反）运动的变化频率，默认值为2.0。当该值大于1时，拖拽层后惯性移动是越来越慢的，值越大，减速效果越明显；当该值小于1时，拖拽层后惯性移动是越来越快的，值越大，加速效果越明显。
