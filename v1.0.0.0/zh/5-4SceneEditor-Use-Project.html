项目使用

导出


  1. 选择菜单栏->导出项目

  2. 设置导出目录

  3. 确认

复制文件

 

将导出目录中的文件全部复制到项目的Resources文件夹中

 

使用

将导出后的项目放到资源文件中。

 

CCScene * newscene  = CCScene::create();
CCNode*pNode=CCJsonReader::sharedJsonReader()->createNodeWithJsonFile("SceneEditorTest/SceneEditorTest.json");
