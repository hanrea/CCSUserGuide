层容器[UIPanel]

图标

描述

UIPanel的类是一个UIWidget子类，它实现了在触摸屏上的层容器。

触摸一个层容器时拦截事件和动作消息发送到目标对象时。设定的目标和行动方法都继承自UIWidget。该控件提供了方法来设置层容器的裁切，宽高，背景色透明度，填充颜色，图片资源等属性。

特性

裁剪

 

隐藏掉控件显示区域外的子控件

 

文件

 

配置一张默认下的图片（png,jpg），控件默认状态下显示。

 

背景色透明度

 

设置背景颜色的透明度，范围0-255。 0表示完全透明，255表示不透明。

 

填充颜色

单色
