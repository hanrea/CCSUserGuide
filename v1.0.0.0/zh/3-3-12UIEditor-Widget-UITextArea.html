文本区[UITextArea]

图标

描述

UITextArea的类是一个UIWidget子类，它实现了在触摸屏上的文本区。

触摸一个文本区时拦截事件和动作消息发送到目标对象时。设定的目标和行动方法都继承自UIWidget。该控件提供了方法来设置文本区的显示宽高，字号，字体，对齐方式，文本内容，是否缩放等属性。

特性

宽度

 

设置控件显示区域宽度。

 

高度

 

设置控件显示区域高度。

 

字号

 

表示字体大小。

 

字体

 

现在只支持宋体，黑体，微软雅黑。

 

水平对齐

 

设置控件的水平对齐方式：

1.HT_Left:文本左对齐

2.HT_Center:文本水平居中

3.HT_Right:文本右对齐

 

垂直对齐

 

设置控件的垂直对齐方式：

1.VT_Top:文本上对齐

2.VT_Center:文本垂直居中

3.VT_Bottom:文本底端对齐

 

文本

 

显示的文本内容。

 

文本缩放

 

控制文本是否缩放。
