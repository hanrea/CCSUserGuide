数字标签[UILableAtlas]

图标

描述

UILableAtlas的类是一个UIWidget子类，它实现了在触摸屏上的数字标签。

触摸一个数字标签时拦截事件和动作消息发送到目标对象时。设定的目标和行动方法都继承自UIWidget。该控件提供了方法来设置标签图片，标签首字符，标签字符宽高，文本内容等属性。
