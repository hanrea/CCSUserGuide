UI Editor简介

概述

UI编辑器支持目前Cocos2d-x所有控件，同时支持碎图合并以及自定义UI，视图化的控件管理能够减少程序的代码量，能够最大限度的加速游戏的开发速度。

 

工作区

  • 渲染面板
  • 对象结构面板
  • UI动画列表面板
  • UI动画帧面板
  • 属性面板
  • 资源面板
  • 画布列表面板
  • 输出面板
  • 控件面板
  • 预览面板
  • 菜单栏
  • 快捷菜单面板
  • 状态栏

控件

  • 按钮
  • 复选框
  • 图片
  • 文本
  • 数字标签
  • 自定义字体
  • 进度条
  • 层容器
  • 滚动层
  • 滑动条
  • 文本区
  • 文本按钮
  • 输入框
  • 拖拽层
