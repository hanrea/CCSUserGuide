文本框[UILable]

图标

描述

UILable的类是一个UIWidget子类，它实现了在触摸屏上的文本框。

触摸一个文本框时拦截事件和动作消息发送到目标对象时。设定的目标和行动方法都继承自UIWidget。该控件提供了方法来设置字号，字体，文本内容等属性。
