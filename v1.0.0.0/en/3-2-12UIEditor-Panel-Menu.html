Menu Bar

Overview

 

The Menu bar contains File, Edit, View, Help and Language.


Functionality

File menu (Alt + F)

New Project (Ctrl + N)

 

Creates a new UI project. If a project is already opened, Cocos Studio prompts you to save the current project. When you set up a new project, specify the project name and location.

 

 

Note:

1：The project name can't contain the following symbols : "?", "`", "\", "*", "<", ">","|""

2. When Cocos Studio is running on Windows XP, do not put the project on your desktop.

3. For all Windows operating systems, do not put the project under the system directory.

 

Open Project(Ctrl+o)

 

Opens the existing project.

 

Close Project

 

Closes the opened project.

 

Save Project(Ctrl-S)

 

Saves the opened project.

 

Save as(Ctrl-Shift+S)

 

Saves the project as a new project（ver 1.1 or later）.

 

Export project

 

Exports the selected canvas. For more information, see UI editor project export.

 

Import template

 

Imports the existing project.

 

Note:

1. Cocos Studio only supports importing the json files which are exported by UI editor.

2. Before you import json files, Cocos Studio will import all the resources which are under the same directory to Cocos Studio.

Import Resource

Import file

1. Imports a file from your hard disk and supports multiple imports.

2. When Cocos Studio encounters a duplicate file name, importing will fail.

 

Import directory

1. Imports a directory from your hard disk.

2. When Cocos Studio encounters a duplicate directory name, importing will fail.

 

Open CCB file

 

Imports CocosBuilder project.

 

Exit (Alt+F4)

 

Closes the editor. If the project is modified, it will prompt you to save the project.

 

Edit(ALT+E)

Undo(Ctrl-Z)

 

Undos the last operation.

 

Redo(ctrl+y)

 

Repeats the last undo operation.

 

Canvas Scale

 

Scales operations on canvas.

 

View（ALT+V）

Hidden/Show Rule

 

Switch render panel's state of the ruler.

 

Clear Guides

 

Remove the reference line in the render panel .

 

Lock Guides

 

Lock the reference line in the render panel.

 

New Guides

 

Create a new reference line.

Note:You can drag and drop ruler to quickly create a reference line.

Window(ALT+W)

Object Structure

 

Opens Object Structure windows

 

Property

 

Opens Property windows.

 

Resource

 

Opens Resource windows.

 

Canvas List

 

Opens Canvas List windows.

 

Output

 

Opens the Output windows.

 

Start page windows

 

Opens the Start page window of the UI editor.

 

Reset windows layout

 

Resets Cocos Studio's window layout to its default value.

 

Help(ALT+H)

View Help(F1)

 

Opens a window displaying the online help documentation.

 

About Cocos Studio.

 

Opens a window displaying information about Cocos Studio.

 

Check New Version

 

Checks for whether there is an existing new version or not.

 

Language(ALT+L)

English

 

Sets English as the editor's default language. It will take effect only after relaunching your editor.

 

Chinese

 

Sets Chinese as the editor's default language. It will take effect only after relaunching your editor.
