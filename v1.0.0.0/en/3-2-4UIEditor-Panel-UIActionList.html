UI Animation List Panel

Overview

 

This panel manages all of your UI animations.

 


Functionality

Modify animation name

 

You can double-click the left mouse button on an animation name or press F12 to modify the animation name.

 

Context Menu

Add Animation

 

Adds a new animation in the current position.

 

Copy Animation

 

Copies the selected animation and pastes it into the end of the animation list.

 

Delete Animation

 

Deletes the selected animation.
