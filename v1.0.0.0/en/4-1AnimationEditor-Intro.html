Animation Editor Introduction

Overview

The Animation Editor supports making and editing the mainstream skeletal animation and the frame animation sequence. It also supports importing Dragon Bones and Flash animations directly.

 

Workspace

  • Resource Manager Panel
  • Properties Panel
  • Resource Panel
  • Animations List Panel
  • Object-Relation Panel
  • Animation Frame Panel
  • Output Panel
  • Preview Panel
  • Menu Bar
  • Quick Tools Panel
