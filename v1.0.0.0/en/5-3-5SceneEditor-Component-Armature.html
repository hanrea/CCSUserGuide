Armature

Icon

 

 

Description

 

The Armature component (ArmatureComponent) contains the function of rendering an armature animation. After binding an armature component, a NULL object will be modified to an armature object.

 

Properties

 

For more information, see Scene component general properties.

 

Feature

Name

 

The ArmatureComponent's name，which can let us get component in programs (getComponent). The component's name can be amended and the default name is CCArmature.

 

ArmatureFile

 

The path of json (with small images) type file or exportjson (with large images) type file.

 

AnimationsList

 

The list of animations in the ArmatureFile.

 

Preview (PlayControl)

  • Play：Play the target animation in AnimationList of this armature.
  • Stop：Stop playing the animation.
