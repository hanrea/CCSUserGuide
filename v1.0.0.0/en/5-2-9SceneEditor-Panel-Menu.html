Menu Panel

Introduction

 

The Scene editor menu bar is a collection of common operations that include four categories: File, Edit, View, Help.


Feature

File(Alt+F)

New Project(Ctrl+N)
  • New game project

    •  

    Create a new game project, the content contains initialization game project. Game project can manage all the UI project, animation, scene project.

     

  • New UI project

    •  

    Create a new UI project, create content under the project folder after CocoStduio ccsprojs directory within the folder.

     

  • New animation project

    •  

    Create a new animation project, create content under the project folder after CocoStduio ccsprojs directory within the folder.

     

  • New scene project

    •  

    After the project is to create a new scene, creating content under the project folder CocoStduio ccsprojs directory within the folder.

     

     

    Note: Before to create the UI / animation / scene project must exist a game project.

     

Open Project(Ctrl+O)

 

Opens an existing project.

 

Save Project(Ctrl+S)

 

Saves the project which is currently being edited.

 

Save As(Ctrl+Shift+N)

 

 

Saves the project as a new project（ver 1.1 or later）.

 

Note: The file name can not contain the following symbols: "？"、"、"、"╲"、"*"、"""、"""、"<"、">"、"|".

 

Export Project

 

Exports the selected canvas as project resources.

 

Import Resources

Importing Files

1. Supports multiple selection Import.

2. Files with duplicate names will not be imported.

 

Import Folder

1. Select folder from disk to import for use in CocoStudio.

2. Folders with duplicate names will not be imported.

 

Exit(Alt+F4)

 

If the project has been modified, the Scene editor will remind the user to save the project.

 

Edit(Alt+E)

Undo(Ctrl+Z)

 

Reverses the last action.

 

Redo(Ctrl+Y)

 

Recovers the reversed action.

 

View（ALT+V）

Hidden/Show Rule

 

Switch render panel's state of the ruler.

 

Clear Guides

 

Remove the reference line in the render panel .

 

Lock Guides

 

Lock the reference line in the render panel.

 

New Guides

 

Create a new reference line.

Note:You can drag and drop ruler to quickly create a reference line.

Window(ALT+W)

Object Struct

 

Opens the Object Structure window.

 

Properties

 

Opens the Properties window.

 

Resources

 

Opens the Resources window.

 

Output

 

Opens the Output window.

 

Start Page

 

Opens the Start Page window.

 

Reset Window Layout

 

Resets the editor to the initial layout.

 

Help(Alt+H)

Open help documents. (F1)

 

Opens online help documents。

 

About Cocostudio

 

Opens CocoStudio information window。

 

Check New Version

 

Checks for updated versions of CocoStudio.

 

Language(Alt+L)

 

Takes effect after restarting the editor if you changed the language.
