Rendering Panel

Overview

 

The Rendering panel provides a rendering widget to support real-time previewing functionality.

 

Functionality

Select

 

Use your mouse to select the entire widget, then copy or delete the widget.


Adjust View

 

Details:

  • At the corners or the 1,2,4, or 5 points of reference as labeled above, you can use your mouse to adjust the size of the widget. You can also scale the widget by simultaneously pressing the Shift key and clicking on the widget to modify the widget to be uniform to scale.
  • By selecting the point of reference as shown in the graphic as point 3, you can rotate the widget.
  • Note that point 6 is an anchor point.

 

Context Menu

Add Object

Button Adds a button to a widget as a sub-widget.
CheckBox Adds a checkbox to a widget as a sub-widget.
Image Adds an image to a widget as a sub-widget.
Label Adds a text field to a widget as a sub-widget.
AtlasLabel Adds a numeric label to a widget as a sub-widget.
BMFontLabel Adds a user-defined font to a widget as a sub-widget.
LoadingBar Adds a loading bar to a widget as a sub-widget.
Panel Adds a layer lock container to a widget as a sub-widget.
ScrollView Adds a scroll view to a widget as a sub-widget.
Slider Adds a slider to a widget as a sub-widget.
TextArea Adds a text area to a widget as a sub-widget.
TextButton Adds a text button to a widget as a sub-widget.
TextField Adds a text field to a widget as a sub-widget.
DragPanel Adds a dragable panel to a widget as a sub-widget.

 

When you select a widget and add a widget through the context menu, CocoStudio will add a widget to your currently selected widget. If you don't select a widget before you launching the context menu to add a widget, CocoStudio will add the widget as a root widget's sub-widget.

Layout Alignment

Center Alignment Aligns the selected widgets to the center.
Left Alignment Aligns the selected widgets to the left.
Right Alignment Aligns the selected widgets to the right.
Top Alignment Aligns the selected widgets to the top.
Bottom Alignment Aligns the selected widgets to the bottom.
Horizontal Alignment Aligns the selected widgets horizontally.
Vertical Alignment Aligns the selected widgets vertically.
Original Alignment Reverts the selected widgets back to their original alignments.

Delete Object

 

Deletes the selected widgets.

 

  1. If no widget is selected, then the menu item will be disabled.
  2. Supports multiple selection.

Copy

Makes a copy of the selected widgets.

 

 

If no widget is selected, then the menu item will be disabled.

 

Paste

Pastes the pre-copied widgets.

 

 

If no widgets were copied before, then the menu item will also be disabled.

 

Edit Animation

Edits the animation of the selected widget.

 

 

Switch to the animation mode first to edit the animation of the widget.
