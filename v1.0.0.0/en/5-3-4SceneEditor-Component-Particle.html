Particle

Icon

 

 

Desctription

 

The Particle component contains the function of rendering a plist type file. After binding a Particle component, a NULL object will be modified to a Particle object.

 

Properties

 

For more information, see Scene component general properties.

 

Feature

Name

 

The ParticleComponent's name，which can let us get component in programs (getComponent). The component's name can be amended and the default name is CCParticleSystemQuad.

 

ParticleFile

 

The path of the plist file.

 

 

Preview (ParticleControl)

  • Play：Plays the particle.
  • Stop：Stops playing the particle.
