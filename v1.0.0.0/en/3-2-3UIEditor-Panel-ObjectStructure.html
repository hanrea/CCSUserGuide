Object Structure Panel

Overview

 

The Object Structure Panel manages all widgets in the UI interface, including child-parent relationship management and hierarchy management. The root object of the Object Structure Panel is a panel.

 

 

Functionality

Show/Hide

 

1. Select the Eye icon on the object tree to toggle visibility by showing or hiding the widget.

2. Use this functionality to set the widget's visible property.


Lock/Unlock

 

1. Select the Circle icon to lock or unlock the widget.

2. When the widget is locked, all associated operations will be disabled.

 

Hierarchy Adjustment

 

1. You can adjust the hierarchy of widgets by selecting the widget with your left mouse button, then dragging and dropping the widgets.

2. After dragging the ImageView widget to the next panel, there will be a blue line between the two panels. When you release your mouse, the ImageView widget will be positioned between the two panels. If you drop the ImageView widget on top of a panel, then there will be a blue circle around the panel. When you release the mouse, the ImageView widget will be added as a subset of the panel.

Layout align

Center Alignment Center aligns the selected widgets.
Left Alignment Left aligns the selected widgets.
Right Alignment Right aligns the selected widgets.
Top Alignment Top aligns the selected widgets.
Bottom Alignment Bottom aligns the selected widgets.
Horizontal Alignment Horizontally aligns the selected widgets.
Vertical Alignment Vertically aligns the selected widgets.
Origin Alignment Reverts the selected widgets back to their original alignment.

 

  1. If no widget is selected or only one widget is selected, the menu item will be disabled.
  2. Alignment will ignore all sub-widgets of the selected widget, because the sub-widget is changing according to its parent.
  3. Logic
    1. First, Cocos Studio computes the whole common rectangle of all widgets to prepare for aligning.
    2. Converts each widget's four points to world coordination.
    3. Calculates each widget's four edges: left side, right side, top side and bottom side.
    4. Calculates the offset. For example, a widget's left side coordination - all widgets' common rectangle's left side = the offset
    5. Subtracts the offset from the widget's coordination to achieve alignment.

Delete Object

 

Deletes the selected object

 

 

1. If no widget is selected, this menu item will be disabled.

2. Supports multiple file deletion.

 

Copy

 

Makes a copy of the selected widget.

 

 

If there is no widget to be selected, the menu item will be disabled.

 

Paste

 

Pastes the pre-copied widget.

 

 

If there is no widget be copied, then the menu item will be disabled.

 

Edit Animation

 

Edits the animation of the selected widget.

 

 

You should switch to the Animation Mode from the menu bar before editing animation.
