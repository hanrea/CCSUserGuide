Animations List Panel

Introduction

 

This panel provides Animations management of the project.


Function

Change the name of the Animation

 

If you want to modify the name of the animation, use the F2 key to open the state modifying the name of the animation.

 

Content Menu

Add Animation

 

Adds a new animation at the current position.

 

Copy Animation

 

Copies a currently selected animation and adds it to the end of the list.

 

Delete Animation

 

Deletes the currently selected animation.
