Text button[UITextButton]

Icon

Description

The UITextButton subclass implements the button for the touch screen. The UITextButton subclass is a subclass of the UIWidget.

The widget provides methods to set the text area for the display's width and height, font size, font, alignment, text content, whether scaling applies, and other attributes.

Feature

Normal state

 

Sets a picture (png, jpg) to display on normal status.

 

Press state

 

Sets a picture (png, jpg) to display on pressed-down status.

 

Disabled state

 

Sets a picture (png, jpg) to display on disabled status.

 

Font styie

 

Supports Arial, bold, and Microsoft Yahei.

 

Text color

 

Sets the text color of the text in the button.

 

Font size

 

The font size of the text button.

 

Text

 

The text button which displays text in the button.
