Sprite

Icon

 

 

Description

 

The Sprite component contains the function of rendering a png texture. After binding to a sprite component, a NULL object will be modified to a png object.

 

属性

 

For more information, please see Scene Component general properties.

 

feature

Name

 

The component's name, which can let us get component in programs (getComponent). The component's name can be amended and the default name is CCSprite.

 

SpriteFile

 

The path of the png file.
