Simulator Selected

Intro

 

 

Cocostudio 1.2.0.0 version began offering the choice of the simulator.You can select the desired simulator according to your needs.

 

Function

EXE Local Path

 

Choose executable file path of the simulator. Default path is the own simulator of cocostudio.

The source code of CocoStudio's simulator has been published to the github.com website. You can modify the program to realize the simulator of youself, add and modify the function you need.

 

Arguments

 

The parameters passed to the simulator.This parameter contains the path information of the current project , it is used to guide the simulator load the scene data.

This parameter is not need to modify by default.But if you custom your own simulator, you can according to your requirement of simulator to fill the parameter.

 

Reset To Default

 

Rest "EXE Local Path" and "Arguments" property to system default value.
