TMXTitledMap

Icon

 

 

Description

 

The TMXTitleMap component contains the function of rendering a tmx type file. After binding to a TMXTitleMap component, a NULL object will be modified to a TMXTitleMap object.

 

Properties

 

For more information, see Scene component general properties.

 

Feature

Name

 

The MapComponent's name, which can let us get component in programs (getComponent). The component's name can be amended and the default name is CCTMXTiledMap.

 

MapFile

 

The path of the tmx file.
